# Code of Conduct

We want a welcoming, inclusive, and harassment-free community.

## Our Standards
- Be respectful and constructive.
- Assume good intent; ask questions before judging.
- No harassment, discrimination, or personal attacks.
- Keep discussions focused on the project.

## Scope
This applies in project spaces (issues, PRs, discussions) and when representing the project.
